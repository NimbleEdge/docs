fun predict(modelId: String, inputs: Array<Any>): ModelOutput
Content copied to clipboard

Runs the Inference pipeline on the device. This is a synchronous predict call and runs the computation on the same primary thread from which it was called. The predict queues the operations if there are multiple predict request for the same model. To optimize resources on the device a single model generates only a single prediction at a time. However multiple models can run in parallel.If you want to call asynchronously see predictAsync.

The method fails gracefully if initialization is not done or not completed. The Error field in ModelOutput will be not null.

NimbleNet.predict("rec_v",arrayOf(arrayOf(1,2,3,4)))
